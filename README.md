# Romance-of-the-Three-Kingdoms-XIV-nju-course-design
2025秋南京大学智能软件工程课程设计

# 游戏玩法：

## 主要类：

### 武将
在武将属性上，每个武将拥有**统率**、**武力**、**智力**、**政治**、**带领的士兵数**以及隐藏的**贪婪**和**忠义**属性。其中，
-   统率影响武将率领的军队的属性；
-   武力影响武将在单挑时的表现；
-   智力影响城池的商业开发情况，同时会稍微降低敌方向你发起单挑的概率；
-   政治影响城池的农业开发情况；
-   贪婪影响武将担任官员时每月所需的俸禄数；
-   忠义影响武将被劝降的概率，忠义越高越难劝降；
-   带领的士兵数最多为1000。

### 城池
城池属性上，每个城池具有**城池商业开发情况**、**城池农业开发情况**、**城池的商业开发官员**、**城池的农业开发官员**、**城内金钱数**和**城内粮草数**这些属性。其中，
-   城池商业开发情况受城池的商业开发官员的智力影响，并影响该城池每月的金钱收入；
-   城池农业开发情况受城池的农业开发官员的政治影响，并影响该城池每月的粮草收入；
-   城池的商（农）业开发官员的智力（政治）影响城池的商业（农业）开发情况；
-   城内金钱数受城池的商业开发情况和城池的武将的影响：商业开发情况越高每月城市收入越高，城池内武将若士兵数未达到最大值会花费金钱募兵，所使用的金钱数取决于要招募的士兵数以及该武将的贪婪属性所影响的俸禄数，如果城池内金钱不够会按照商业-农业-募兵的顺序优先为能发俸禄的人发放俸禄，本月没有俸禄的人就不执行对应操作；
-   城内粮草数受农业开发情况以及士兵数影响，农业开发情况越高，每月粮草收入越高，士兵越多每月粮草支出越多，如果粮草不足以匹配士兵数量，未发放粮草的部分士兵会进行逃亡（即城池内部分武将的士兵数减少）。

### 军队
军队属性上，每个军队有**阵型**、**军队主将**和**士兵数**这几个显式属性与**攻击**和**防御**这两个隐式属性。其中，
-   阵型：包括：
    +   投石阵，借助投石车进行远距离攻击，面对方圆阵时可以直接破坏对方阵法从而造成更多伤害；
    +   锋矢阵，精锐的士兵所组成的突围队伍，适合突袭拥有投石车无法快速移动的笨重队伍，容易触发与对方主将的单挑，面对方圆阵时不仅无法突破阵型反而会受到额外损伤；
    +   方圆阵，由士兵罗列组成的铁桶阵，可以有效阻挡锋矢阵的冲击，但在面对投石车时会因为阵型的机动过低导致长时间只受到投石车攻击而无法对对方造成有效伤害；
-   军队主将，统帅军队的主将，其统帅将会影响军队的攻击与防御，主将的武力与智力将会影响单挑概率，**单挑获胜方军队会在本轮作战中获得额外的全方面增益**；
-   士兵数，军队主将所带领的士兵数。

## 基本玩法：

玩家将扮演蜀汉势力的君主刘备，一统天下，匡扶汉室。具体来说，玩家每回合有8次行动的机会，玩家可以点击城市与城市互动，进行**探索**、**劝降**、**出兵作战**、**运输**、**买卖粮食**、**调遣武将**、**设置官员**与**查看城市情报**：
-   *探索*：玩家消耗一次行动机会，对该城市进行探索，有几率发现在野武将或者获得额外的粮草或金钱；
-   *劝降*：当该城市中关有武将时可以消耗一次行动机会进行劝降，劝降的成功率取决于对方的忠义程度以及该武将被关押的回合数，需要注意的是，**关押的越久，囚犯逃脱概率越高**；
-   *出兵作战*：消耗一次行动机会向邻近的敌方城市进攻，只有与敌方城市接壤的前线城市可以派遣若干名士兵数大于0的武将出兵作战；
-   *运输*：消耗一次行动机会从一个己方城市向另一个己方城市运输钱粮；
-   *买卖粮食*：消耗一次行动机会在一个城市购买或贩卖粮食；
-   *调遣武将*：消耗一次行动机会将本城市的一名武将调往其他己方城市；
-   *设置官员*：消耗一次行动机会为己方的某个城市设置商业或者农业官员进行商（农）业开发；
-   *查看城市情报*：查看一个己方或者敌方城市中的武将信息，不消耗行动次数。

## 出兵作战：

派遣武将攻击敌方城市或者我方城市被敌方攻击时进入作战，作战采用回合制：
由进攻方先发动一次进攻，选择一支己方军队与出阵阵型并挑选将要挑战的敌方军队开展作战，敌方选择完毕阵型后两军开战，战斗以某一方军队士兵数归零中止，当士兵数归零时，军队的主将有概率直接被敌方俘获；
一次战斗结束后，战斗与防守顺序互换，由被攻击方选择一支己方军队与出阵阵型并挑选将要挑战的敌方军队开展第二轮作战，两军重复此过程直至进攻方主动撤军或者有一方没有任何可以作战的军队。
需要注意的是，每轮战斗开始前，两军主将会有概率触发单挑，单挑概率受双方主将的武力与智力影响。
